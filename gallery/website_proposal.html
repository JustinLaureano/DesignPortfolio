Overview

This website and proposal was deisgned for personal trainer Jonathan Schoonover while I attended Owens Community College. The website reflects the services offered by Jonathan and the proposal outlines all the details for putting the website together.

The color sheme was influenced by Jonathan's Alma Mater, BGSU, and the website was designed using Adobe Dreamweaver and Flash as dictated by the syllabus for the project. I later recreated the website using Bootstrap 3 for a responsive layout.
